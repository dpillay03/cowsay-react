# Cowsay React
## Developer: Daniel Pillay

## Description
A random text generator that returns random quotes with cowsay.

## Install
1. Clone repository
1. Run `npm install`
1. Run `nodemon` or `node index.js` if nodemon is not installed globally.

## Technologies
1. Languages: JavaScript
1. `babel-core`
- `babel-loader`
- `babel-preset-env`
- `babel-preset-react`
